# 🥗 NutriFlow – Dietician Management System

🚀 AI-powered nutrition management platform designed to connect **Patients, Dieticians, and Administrators** in a single ecosystem.

🔗 **Repository:** https://github.com/GuhanSenthil/NutriFlow-Dietician-Management-System.git

---

## 📌 Overview

NutriFlow is a full-stack web application that streamlines diet planning, patient monitoring, and healthcare communication.  
It enables structured meal tracking, real-time interaction, and data-driven insights for better health outcomes.

---

## ✨ Key Features

### 👤 Patient Dashboard

- 📊 BMI & Health Progress Tracking  
- 🍽️ Food Diary & Symptom Logging  
- 📅 Appointment Booking  
- 💬 Chat with Dieticians  
- 💳 Payment & Billing History  
- 📚 Articles & Recipe Access  

---

### 🧑‍⚕️ Dietician Dashboard

- 👥 Patient Request Management  
- 📝 Create Diet Plans & Content  
- 💬 Real-time Communication  
- 📅 Appointment Scheduling  

---

### 🛠️ Admin Dashboard

- 👤 User & Role Management  
- 📊 Analytics & Platform Monitoring  
- 💰 Revenue Tracking  
- 📢 Global Announcements  

---

## 🧠 Tech Stack

### Frontend
- React 19 + TypeScript  
- Tailwind CSS  
- Recharts  
- Vite  

### Backend (In Progress)
- Node.js  
- Express.js  
- REST API  

### Data Handling
- Mock API (In-Memory Database)  
- Planned: Firebase Firestore
